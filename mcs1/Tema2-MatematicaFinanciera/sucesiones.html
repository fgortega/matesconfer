Sucesiones

Una sucesión es un conjunto ordenado de números (u otros objetos), generados de acuerdo a una regla.
Por ejemplo:
· 1, 2, 3, 4, 5, …
· 2, 4, 6, 8, 10, …
· 1, 4, 9, 16, 25, …
· 32, 16, 8, 4, 2, …
· 1, 1, 2, 3, 5, 8, 13, …
Cada número de la sucesión se llama término, y le denotamos ai. Llamamos término general de la sucesión, an, a la expresión que representa cualquier término de la sucesión.
Por ejemplo:
·1, 2, 3, 4, 5, …  an = n, entonces a100 = 100
· 2, 4, 6, 8, 10, …  an = 2n, entonces a25 = 50
· 1, 4, 9, 16, 25, …  an = n2, entonces a12 = 144
· 32, 16, 8, 4, 2, …      an = 26-n, entonces a10 = 2-4 = 1/16
Ejercicio. Encuentra el término general de estas sucesiones:
a) 1/2, 2/3, 3/4,...
b) -1, 1, -1, 1, -1, 1,...
c) 3, 6, 9, 12, 15,...
Soluciones: a) n/(n+1); b) (-1)n; c) 3n

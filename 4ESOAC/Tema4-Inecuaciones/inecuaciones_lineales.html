Inecuaciones lineales

Para resolver una inecuación de primer grado, tenemos que obtener una inecuación equivalente hasta que despejemos la incógnita:
Ejemplo 1:

  2x + 3 ≤ 4x + 7         2x + 3 – 3 ≤ 4x + 7 – 3

  2x – 4x ≤ 4x + 4 – 4x       - 2x ≤ 4       x ≥ -2

  x є [-2,∞)

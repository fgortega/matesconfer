Factor común

Lo contrario de la propiedad distributiva es sacar factor común:

  a · b + a · c = a·(b + c);   a · c + b · c = (a + b)· c

Ejemplos:

a2 + ab = a · a + a · b = a ·(a + b)

4x3 + 8x2 - 2x = 2 · 2 · x · x · x + 2 · 2 · 2 · x · x - 2 · x = 2x ·(2x2 + 4x - 1)

 

 

Ejercicio. Saca factor común:

a) 3xy + 6xy2 - 12x2y2

b) 16x3 + 12x2 + 28x

c) 18abc + 12ab2c3 - 24abc5 - 6ac

 

 

 

 

Soluciones: a) 3xy·(1 + 2y - 4xy); b) 4x·(4x2 + 3x + 7); c) 6ac·(3b + 2b2c2 - 4bc4 - 1)

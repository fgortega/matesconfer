jerarquía de las operaciones

Sigue siempre el orden correcto (o jerarquía) de las operaciones:

- Primero paréntesis

- Segundo divide y multiplica (de izquierda a derecha)

- Lo último, las sumas y las restas

Ejemplo:

3 + 2·(5·3 - 4:2) = 3 + 2·(15 - 2) = 3 + 2·13 = 3 + 26 = 29

Sistema de numeración romano

 Los romanos usaban estos caracteres:
   ''I'' = uno         ''V'' = cinco     ''X'' = diez     ''L'' = cincuenta     ''C'' = cien    ''D'' = quinientos    ''M'' = mil

Estos caracteres se usaban de forma aditiva, exceto para 4, 9, 40, 90,... En estos casos se resta el número menor (en la izquierda) del mayor (en la derecha)
 
  Ejemplo:         

  14 = XIV            2009 = MMIX      1459 = MCDLIX

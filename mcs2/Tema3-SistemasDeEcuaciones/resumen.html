Resumen

Una ecuación de grado 1 con una o varias incógnitas se llama ecuación lineal.

Todas las ecuaciones lineales tiene esta forma:         a1x1 + a2x2 + …..+ anxn = b

donde a1,a2,…, an,b son números reales y x1, x2, ….., xn son incógnitas o variables

Si b = 0 se llama ecuación homogénea.

Una solución de la ecuación es un conjunto de valores de las variables, x1 = k1, x2 = k2,…., xn = kn, los cuales convierten la ecuación en una igualdad numérica.

Un sistema de m ecuaciones lineales con n incógnitas es un conjunto de m ecuaciones lineales con n incógnitas que deben tener solución común. Tiene esta forma


donde x1,x2,…….xn son las variables y aij, bi son números reales i = 1,2….m, j = 1,2,….n
Una solución de un sistema es una solución de todas las ecuaciones.

Resolver un sistema es encontrar todas sus soluciones.

Dependiendo del número de soluciones, podemos clasificar los sistemas en:

–Sistemas compatibles: si existe solución. Pueden ser:

  • Sistemas compatibles determinados: con sólo una solución
  • Sistemas compatibles indeterminados: con infinitas soluciones

–Sistemas incompatibles: sin solución

NOTA#1: Un sistema homogéneo siempre tiene solución
Una ecuación degenerada tiene la forma:

                 0x1 + 0x2 + ….. + 0xn = b

Entonces:

–Si b = 0, se llama ecuación trivial

–Si b ≠ 0, se llama ecuación absurda

NOTA#2: si existe una ecuación absurda el sistema es incompatible

Dos ecuaciones son equivalentes si tienen las mismas soluciones.

Dos sistemas son equivalentes si tienen las mismas soluciones.

Para obtener sistemas equivalentes, tenemos que hacer las siguientes operaciones elementales:

- Cambiar dos ecuaciones:  Ei ↔ Ej

- Sustituir una ecuación por una combinación lineal de ecuaciones:

 Ei↔ k1E1+k2E2+…+kiEi+…kmEm, ki ≠ 0, kj son números reales, j = 1, 2, ….m

El hecho de que un sistema se pueda representar en forma matricial implica que también se puede resolver como si fuera una ecuación matricial.

Igualdades algebraicas

Una ecuación es una igualdad algebraica que es cierta sólo para ciertos valores de las variables. Estos valores se llaman soluciones de la ecuación.

Una identidad es una igualdad algebraica que es cierta para todos los valores de las variables.

Ejemplos:

  • a + a = 2a es una identidad
  • x + 7 = 9 es una ecuación. x = 2 es la solución
  • x + y = 5 es una ecuación. x = 8, y = -3 es una solución

 

Ejercicio. Decide si las siguientes igualdades son identidades o ecuaciones:

a) x + y + z = 28

b) x·(x+5) = x2 +5x

c) x2 - 5x + 4 = x2 +25x

d) (x+1)2 = x2 + 2x +1

 

 

Soluciones: a) ecuación; b) identidad; c) ecuación; d) identidad

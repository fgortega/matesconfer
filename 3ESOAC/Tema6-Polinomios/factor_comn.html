Factor común

Lo contrario a la propiedad distributiva es sacar factor común.


           a · b + a · c = a·(b + c); a · c + b · c = (a + b)· c


Ejemplos:


a2 + ab = a · a + a · b = a ·(a + b)


4x3 + 8x2 - 2x = 2 · 2 · x · x · x + 2 · 2 · 2 · x · x - 2 · x = 2x ·(2x2 + 4x - 1)

 

 

Ejercicio. Saca factor común en los siguientes polinomios:

a) 2x7 - 36x5 + 12x2 + 16x =

b) 3ab2 + 12ab3 - 6a2b5 =

c) 12x2y5 - 6x2y2 =

 

 

Soluciones: a)  2x·(x5 - 18x4 + 6x + 8); b) 3ab2·(1 + 4b - 2ab3); c) 6x2y2·(2y3 - 1)
